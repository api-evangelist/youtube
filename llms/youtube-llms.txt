# YouTube APIs

> YouTube (Google) APIs provide programmatic access to YouTube data including videos, playlists, channels, user interactions, live streaming, analytics, captions, and embedded player controls. Access is over REST on Google API hosts, authenticated with OAuth 2.0 (accounts.google.com) plus API keys for public read data, and metered by a daily quota-unit system. Generated by API Evangelist from the catalog on 2026-06-20; not published by YouTube.

## APIs
- [YouTube Data API v3](https://developers.google.com/youtube/v3/docs): videos, channels, playlists, search, subscriptions, comments, captions, thumbnails, members. Base https://www.googleapis.com/youtube/v3
- [YouTube Analytics API v2](https://developers.google.com/youtube/analytics): custom reports of views, watch time, and revenue by dimension. Base https://youtubeanalytics.googleapis.com/v2
- [YouTube Reporting API v1](https://developers.google.com/youtube/reporting): bulk daily CSV report datasets. Base https://youtubereporting.googleapis.com/v1
- [YouTube Live Streaming API](https://developers.google.com/youtube/v3/live/docs): liveBroadcasts, liveStreams, liveChatMessages, moderators. Base https://www.googleapis.com/youtube/v3
- [YouTube Content ID API](https://developers.google.com/youtube/partner): partner rights management (assets, claims, policies). Base https://www.googleapis.com/youtube/partner/v1
- [YouTube IFrame Player API](https://developers.google.com/youtube/iframe_api_reference): embed and control a player via JavaScript.
- [YouTube oEmbed](https://www.youtube.com/oembed): oEmbed-compliant embed metadata.
